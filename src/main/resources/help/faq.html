❓ 常见问题

Q: 如何导入现有模组项目？
A: 点击“文件 → 导入项目”，选择 .efmod 文件或包含 build.gradle 的模组文件夹。
Q: 生成的模组无法在游戏中运行？
A: 请检查：
- Minecraft 版本和 Forge 版本是否匹配。
- 物品/方块的内部ID是否唯一且符合规范（小写字母、数字、下划线）。
- 纹理图片是否为 PNG 格式且路径正确。
- 项目是否已成功执行 Gradle 构建（可在项目目录运行 gradlew build）。
Q: 如何添加自定义模型？
A: 在实体编辑器中设置模型路径，支持 Java 模型（需编写代码）或 GeoJSON 模型（需配合 GeckoLib）。本工具暂不自动生成模型代码，需手动补充。
Q: 软件打开时提示“无法检查更新”？
A: 请确保网络畅通，GitHub API 有时会被限制。您也可以手动访问 GitHub Releases 查看新版本。
Q: 如何批量导入物品？
A: 使用 CSV 文件（第一列为ID，第二列显示名称，第三列最大堆叠数，第四列纹理路径），然后点击“批量导入物品”。
